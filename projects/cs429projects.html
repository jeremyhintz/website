Computer Architecture

Projects for CS 429: Computer Architecture


Class Overview

CS 429 covers how computers are organized and how primitive building blocks can be used to create a computing system. It introduces the modern microprocessor memory hierarchy and other aspects of contemporary computing systems.

M0 Assembly Language

For this project, I was required to write an interpreter in C for a basic assembly language called M0 that did adds, jumps, and other basic commands. The program was required to output equivalent C code for the M0 commands.

Binary Bomb Lab

For the next project, we were given object code files, which when ran prompted us for 6 password strings in order to defuse a bomb. If you enter a string wrong, the bomb explodes (and you lose points on your grade). In order to determine what the correct strings are, you must delve into the assembly code using GDB and look for clues.
