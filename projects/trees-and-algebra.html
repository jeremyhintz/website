Trees and Algebra

A Basic Symbolic Calculator Using Tree Recursion


The Assignment

This assignment was done for Gordon Novak's CS 314 class during the Fall 2013 semester. It asks you to write a recursive function that attempts to solve an equation e for variable v. We assume that any variable occurs at most once in e, and that the equation initially has a left-hand side that is a single variable. The basic idea is to test for equations that are already solved or unsolvable, otherwise, search for a solution by applying laws of algebra to transform the equation into different forms.
