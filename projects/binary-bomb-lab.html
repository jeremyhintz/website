Binary Bomb Lab

Defusing a Boomb with GDB


The Assignment

This lab was done in Bill Young's CS 429 class in Spring 2013. We were all given different object code files, which when ran prompted us for 6 strings in order to defuse a bomb. If you enter a string wrong, the bomb explodes (and you lose points on your grade). In order to determine what the correct strings are, you must delve into the assembly code using GDB and look for clues.
